== Description ==
Adds a page template, shortcode, and widget to Wordpress that asks for visitor opinion, then sends them to either \'Share\' a review, or \'Give Feedback\'. Supports Gravity Forms for Feedback.

== Installation ==
Follow the steps below to install the plugin.

• Upload the opinion-share folder and all contents to /wp-content/plugins
• Activate the plugin through the \'Plugins\' menu in WordPress
• Go to Settings->Opinion Share to set Options
• Go to Appearance->Widgets and activate Social Shares
• Go to Pages and select Opinion Share Page under Template
• Optionally, Go to Forms->New Form to create a Feedback Form (Requires Gravity Forms to be installed).
